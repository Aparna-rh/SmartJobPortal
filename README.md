# Smart Job Portal

A full-stack job portal built using Spring Boot, React, MySQL, Railway, and Vercel.

## Features
- User Registration & Login
- Search Jobs
- Internship Listings
- Apply for Jobs
- My Applications
- Notifications
- Admin Dashboard

## Tech Stack
- Frontend: React.js
- Backend: Spring Boot
- Database: MySQL
- Deployment: Railway + Vercel
